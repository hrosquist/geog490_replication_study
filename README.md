# GEOG 490 Replication Study
This project replicates a landscape classification study for the Austin metropolitan area using Census data and Hanberry style population density thresholds.

## Project Summary
The analysis uses 2020 Census tract population data and ACS demographic variables to classify Austin area census tracts into landscape categories, including exurban, suburban, and urban types.

## Outputs
- Austin metropolitan landscape classification map
- Median household income chart by landscape type
- College degree attainment map
- Urban and suburban population pyramids

## Data Sources
- U.S. Census Bureau Decennial Census 2020
- American Community Survey 2016–2020 5-year estimates
- TIGER/Line Core Based Statistical Areas
